Dengan menggunakan Isikuy, kamu menyetujui syarat dan ketentuan berikut. Mohon dibaca sebelum menggunakan aplikasi.

Catatan penting: Isikuy adalah proyek belajar (non-komersial). Semua transaksi yang terjadi di dalam aplikasi bersifat simulasi dan tidak melibatkan uang sungguhan, payment gateway, maupun provider PPOB resmi.

1. Tentang Layanan

Isikuy adalah aplikasi demo yang mensimulasikan layanan isi pulsa, kuota internet, token listrik PLN, top up e-wallet, dan top up game. Aplikasi ini dibuat untuk tujuan edukasi dalam mempelajari pengembangan web dan integrasi layanan pihak ketiga.

2. Akun Pengguna

Untuk menggunakan Isikuy, kamu perlu masuk menggunakan akun Google. Kamu bertanggung jawab untuk menjaga keamanan akun Google kamu sendiri.

3. Sifat Simulasi

Seluruh saldo, harga produk, dan hasil transaksi yang ditampilkan dalam aplikasi adalah data simulasi untuk keperluan demonstrasi. Tidak ada:

  • Pengiriman pulsa, kuota, token listrik, atau item game yang sesungguhnya
  • Pemrosesan pembayaran uang asli
  • Jaminan layanan komersial dalam bentuk apapun

4. Penggunaan yang Wajar

Kamu setuju untuk tidak menyalahgunakan aplikasi ini, termasuk namun tidak terbatas pada upaya mengakses data pengguna lain, atau menggunakan aplikasi untuk tujuan yang melanggar hukum.

5. Tidak Ada Garansi

Aplikasi ini disediakan "sebagaimana adanya" tanpa jaminan apapun, baik tersurat maupun tersirat. Sebagai proyek belajar, aplikasi dapat mengalami perubahan, gangguan, atau penghentian layanan kapan saja tanpa pemberitahuan sebelumnya.

6. Perubahan Layanan

Kami dapat mengubah, menghentikan, atau memperbarui fitur aplikasi ini kapan saja sesuai kebutuhan pengembangan.

7. Kontak

Pertanyaan terkait syarat dan ketentuan ini dapat disampaikan melalui email yang tercantum pada profil developer di Google Cloud Console aplikasi ini.
